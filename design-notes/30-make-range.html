#30: Make Range (Change)


Version 1, January 1994
Copyright (c) 1993-1994, Apple Computer
This design note defines (make <range> ...) to create a range.
Define (make <range> ...) to create a range. The permitted initargs are exactly those currently documented with the range function on p. 118 of the Dylan book.

In the documentation of the <range> class on p. 112, remove the statement "A <range> is normally created by a call to the range function, rather than by a direct call to make."
